Lesson Configuration Template

This template describes the structure of a lesson configuration JSON file. Each lesson includes a prompt, a correct translation (solution), and distractors. The distractors are space-separated strings, containing words similar in meaning or sounding to the words in the solution.

Template Structure

{
    "lessons": [
        {
            "prompt": "Original sentence to translate",
            "solution": "Correct translation of the prompt",
            "distractors": "distractor1 distractor2 distractor3"
        },
        {
            "prompt": "Another translation practice",
            "solution": "Correct translation of the prompt",
            "distractors": "distractorA distractorB distractorC"
        }
        // Add more lessons as needed
    ]
}

Example

{
    "lessons": [
        {
            "prompt": "He runs every morning.",
            "solution": "Er läuft jeden Morgen",
            "distractors": "geht Abend Nacht"
        },
        {
            "prompt": "Where is the nearest hospital?",
            "solution": "Wo ist das nächste Krankenhaus?",
            "distractors": "Klinik Notfall Zentrum"
        },
        {
            "prompt": "I can't hear you.",
            "solution": "Ich kann dich nicht hören",
            "distractors": "sehen sprechen fühlen"
        }
        // Add more lessons as needed
    ]
}

Copy the template structure above and replace the placeholders with your own prompts, solutions, and distractors to generate new lessons using an LLM.
