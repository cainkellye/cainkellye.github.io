Lesson Configuration Template

This template describes the structure of a lesson configuration JSON file. It supports bidirectional lessons with language A and B properties. Each lesson can be practiced in both directions (A→B and B→A). The noiseA and noiseB properties contain distractor words for each language. If noise is missing, it will be automatically generated from other lessons.

Template Structure

{
    "title": "Lesson Set Title",
    "langA-B": ["lang_code_A", "lang_code_B"],
    "lessons": [
        {
            "A": "Sentence in language A",
            "B": "Translation in language B",
            "noiseA": "distractor1 distractor2 distractor3",
            "noiseB": "distractor_a distractor_b distractor_c"
        },
        {
            "A": "Another sentence in language A",
            "B": "Another translation in language B"
            // Note: noiseA and noiseB are optional - will be auto-generated if missing
        }
    ]
}

Example

{
    "title": "Spanish-English Basics",
    "langA-B": ["es", "en"],
    "lessons": [
        {
            "A": "¿Cómo estás?",
            "B": "How are you?",
            "noiseA": "Dónde Qué hora es",
            "noiseB": "Who How old"
        },
        {
            "A": "Buenos días.",
            "B": "Good morning."
        },
        {
            "A": "Me llamo Juan.",
            "B": "My name is Juan.",
            "noiseA": "llamar nombre soy",
            "noiseB": "called named am"
        }
    ]
}

Features

  • Bidirectional Practice: Each lesson can be practiced in both directions (A→B and B→A)
  • Automatic Noise Generation: If noiseA or noiseB are missing, distractor words are automatically generated from other lessons
  • Direction Switching: Use the direction button to switch between translation directions
  • Language-Specific Vocabulary: Vocabulary buttons generate word lists for each language

Copy the template structure above and replace the placeholders with your own sentences, translations, and distractor words, or generate new lessons using an LLM.
