Lesson Configuration Template

This template describes the structure of a lesson configuration JSON file. Each lesson includes a prompt, a correct translation (solution), distractors, and a language setting. The distractors are space-separated strings, containing words similar in meaning or sounding to the words in the solution. The lang property must be set to "en;pt" if the prompt is in English or "pt;en" if the prompt is in Portuguese.

Template Structure

{
    "lessons": [
        {
            "prompt": "Original sentence to translate in german",
            "solution": "Correct translation of the prompt in english",
            "distractors": "distractor1 distractor2 distractor3",
            "lang": "de;en"  // "prompt lang code;solution lang code"
        },
        {
            "prompt": "Another translation practice in english",
            "solution": "Correct translation of the prompt in german",
            "distractors": "distractorA distractorB distractorC",
            "lang": "en;pt"  //  "prompt lang code;solution lang code"
        }
        // Add more lessons as needed
    ]
}
